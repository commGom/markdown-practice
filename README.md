# 표(table)

| 값       | 의미              | 기본값 |
| -------- | ----------------- | ------ |
| static   | 기준 없음         | 0      |
| relative | 요소 자신         | x      |
| absolute | 위치 상 부모 요소 | x      |
| fixed    | 뷰포트            | x      |
